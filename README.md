# Boot Agnóstico v2
Repositorio oficial — arquitectura central del sistema y documentos canónicos.

## Propósito
El Boot Agnóstico v2 es un sistema modular compuesto por máquinas semánticas que operan bajo un canon estricto. 
Este repositorio contiene la estructura física principal y los documentos que definen su arquitectura, reglas y evolución.

## Estructura inicial
- 01_governance — gobernanza, validación, drift detection, hashing
- 02_automatica — automatización, n8n, flujos
- 03_creativa — semántica creativa, plantillas, variables
- 04_financiera — señales económicas y retorno
- 05_cognitiva — análisis extendido y modelos cognitivos
- archivo_historico — eras previas (sin YAML activo)
- verticales — verticales que integrará el Boot

## Estado actual
Canon v2 en construcción. Documento rector y documentos técnicos se crearán progresivamente.
